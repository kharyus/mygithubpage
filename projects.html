Spritesheet Manipulator


It detects sprite's boundaries in a sprite sheet, and then outputs them to separate files. Will also have a button to generate a .json containing the positions to know where are the limits of each sprite. This software is particularly useful for sprite sheets containing no pattern in how they are laid out.

Forum discussion

Made using C++11, SFML(Graphic Library) and TGUI. Currently have Windows 64 bit binaries, and I am working on implementing Linux support. Released under GPL license.
